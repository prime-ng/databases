# Important Points to Add for Enhancement

## 1. heck if any Student left in the School who need to be Dropped
  - Helper will take the Boarding Attendance for Drop Trip.
  - After that he/she will click the Start button to start Drop Trip.
  - System will check if any Student missing in the Drop Attendance by compairing with all the students who were picked up in the Pickup Trip of the same Route.
  - If any Student missing in the Drop Attendance then System will raise an alert to the Helper to take the Boarding Attendance for that Student.
  - If that Student has left the school already then he/she would be registered as Left in the system.
