## SUMMARY OF DASHBOARDS

| Dashboard | Primary Users | Purpose | Key Metrics |
|-----------|--------------|---------|-------------|
| **Admin/Management** | Principal, Management | High-level overview | Total collection, outstanding, class-wise status, top defaulters |
| **Accounts/Cashier** | Accountants, Cashiers | Daily operations | Today's collection, pending tasks, quick collection counter |
| **Parent/Student** | Parents, Students | Self-service | Fee summary, installments, payment history, receipts |
| **Class Teacher** | Class Teachers | Class monitoring | Class-wise pending, defaulter list, concession requests |
| **Analytics** | Management, Decision Makers | Strategic insights | Risk prediction, trends, recommendations |

Each dashboard is designed with:
- **Role-based access** (users see only relevant data)
- **Real-time updates** (where applicable)
- **Drill-down capability** (click through to detailed reports)
- **Action buttons** (take immediate action from dashboard)
- **Visual indicators** (color-coded status, progress bars, trend arrows)
