**sch_settings table detail**
-----------------------------

Key - `Subj_Group_will_be_used_for_all_sections_of_a_class` Value - True/False (Boolean)
    (This key will be used to define whether School will have same Subject Groups for all the Sections of every Class?)
Key - `Allow_extra_student_in_vehicale_beyond_capacity` Value - True/False (Boolean)
    (This key will indicate if school allow extra students beyond vehicle capacity or not?)
Key - `Allow_only_one_side_transport_charges` Value - True/False (Boolean)
    (This Key will indicate if school allow one side fare to be charged or not?)
Key - `Allow_different_pickup_and_drop_point` Value - True/False (Boolean)
    (This Key will indicate if school allow a student to opt different pickup & drop point and pay fare accordingly?)
Key - `trip_usage_needs_to_be_updated_into_vendor_usage_log` Value - True/False (Boolean)
    (This key will be used to define whether School has taken Vendor Module and Entry into Vendor usage log is required or not?)
Key - `Avreage_no_of_student_per_section` Value - 25 (Numeric)
    (This will Capture average student per section)
Key - `Minimum_no_of_student_per_section` Value - 10 (Numeric)
    (This will Capture minimum required student per section)
Key - `Maximum_no_of_student_per_section` Value - 40 (Numeric)
    (This will Capture maximum required student per section)
Key - `section_of_a_class_has_home_room` Value - True/False (Boolean)
    (This key will be used to define whether School has a Room Assigned to Every Section of a Class and Teacher will change Rooms according to the Class they need to teach)
Key - `teacher_has_home_room` Value - True/False (Boolean)
    (This key will be used to define whether School has a Room Assigned to Every Teacher and Student will change the Room according to the Subject Teacher)
Key - `library_media_storage_folder` Value - "" (String)
    (This key will be used to define the folder path where library media files will be stored)
